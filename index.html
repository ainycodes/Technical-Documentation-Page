Introduction

Welcome to the Bootstrap Documentation, your guide to creating responsive and stylish web applications with ease.

Grid System

The Bootstrap grid system allows you to create flexible and responsive layouts. It is based on a 12-column grid, making it easy to organize content for different screen sizes.

Here is a simple example of a grid layout:

<div class="container"> <div class="row"> <div class="col">Column 1</div> <div class="col">Column 2</div> <div class="col">Column 3</div> </div> </div>
Typography

Bootstrap provides a set of utility classes to enhance and style your typography. You can easily adjust text alignment, font weight, and other typographic elements.

For bold text, simply use the class .font-weight-bold:

<h1 class="font-weight-bold">Bold Heading</h1>
Components

Bootstrap comes with a rich set of pre-designed components that you can easily integrate into your projects. These include navigation bars, buttons, forms, and more.

Let's add a button to your page using Bootstrap:

<button class="btn btn-primary">Click me</button>
Utilities

Bootstrap utilities offer a set of handy classes for quick adjustments without writing custom CSS. These classes cover margins, padding, colors, and more.

For example, to add top margin to an element, use the class .mt-4:

<div class="mt-4">This has a margin-top of 4 units</div>
Theming

Bootstrap allows you to customize the look and feel of your project with ease. From colors to spacing, you have control over various aspects of the framework.

Let's explore how to customize Bootstrap's primary color:

<style> :root { --bs-primary: #007bff; } </style>
